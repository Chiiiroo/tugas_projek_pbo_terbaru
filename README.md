## 🛠️ Fitur & Implementasi PBO

Proyek ini mengimplementasikan konsep dasar PBO (Object Oriented Programming) menggunakan PHP:

- **Class & Object:** Implementasi `class Pelanggan` untuk enkapsulasi data.
- **Methods:** Penggunaan method untuk menampilkan output data secara dinamis.
- **Responsive Design:** Tampilan form yang bersih dan user-friendly sesuai instruksi tugas.

## 🚀 Cara Menjalankan Secara Lokal

1. Pastikan XAMPP (Apache) sudah aktif.
2. Clone repository ini ke folder `htdocs`.
3. Akses melalui browser: `localhost/tugas_projek_pbo_terbaru/index.php`.
